## Description
This is my personal portfolio website built using **Express.js**, **EJS**, and **Bootstrap**.  
It includes four pages: **Home**, **About**, **Projects**, and **Contact**.  
The website demonstrates my web development and cybersecurity interests.

## Features
- Node.js & Express.js backend
- EJS templating engine
- Bootstrap responsive design
- Custom CSS gradient theme
- Navigation bar and footer using EJS partials
- Profile image and personal info

## How to Run
1. Clone or download this repository  
2. Open it in VS Code  
3. Run `npm install`  
4. Start the server with `npm start`  
5. Visit `http://localhost:3000` in your browser

## Technologies Used
- Node.js  
- Express.js  
- EJS  
- Bootstrap  
- HTML5 / CSS3
